      注意魔女集會以及圈子 #

      兩名十幾歲的女巫其樂融融地聚在她們的小閣樓裡，面對面坐著。儀式鞏固了她們的契約，把對對方的愛捆成作為同伴的最後一個結。

      魔女集會是我們圈子特有的，相互之間的精神上的連結。很多人會和家人、朋友，或同盟，舉行魔女集會。一些魔女集會是涉及魔法的，亦有很多是建立在共識和尊重之上。

      • 當有人舉行魔女集會，問：「你共享什麼目的？」
      • 當有人從魔女集會中脫離，問：「誰會因此最受傷？」
